# 📱 Описание телеграм-бота

## Описание бота (Short Description в BotFather)
```
🚰 Система мониторинга водоснабжения с авторизацией
```

## Длинное описание бота (About в BotFather)
```
🚰 Система мониторинга водоснабжения

Этот бот предоставляет доступ к системе мониторинга водоснабжения и позволяет отслеживать:

🔔 Алерты и уведомления по водоснабжению
📊 Статистику потребления воды
🏠 Информацию о домах и ЦТП
🗺️ Поиск объектов по координатам

⚠️ Для использования бота необходимо зарегистрироваться командой /register

Доступные команды:
• /start - начало работы
• /register - регистрация нового пользователя
• /help - справка по командам
```

## Инструкции для настройки в BotFather

### 1. Short Description (120 символов макс.)
```
🚰 Система мониторинга водоснабжения с авторизацией
```

### 2. Description (512 символов макс.)
```
🚰 Система мониторинга водоснабжения

Отслеживайте алерты, статистику потребления и информацию о домах и ЦТП.

⚠️ Требуется регистрация командой /register
```

### 3. Commands (список команд для BotFather)
```
start - Начать работу с ботом
register - Регистрация нового пользователя  
login - Вход в систему
logout - Выход из системы
profile - Просмотр профиля
help - Справка по командам
alerts - Получить текущие алерты
subscribe - Подписаться на уведомления
unsubscribe - Отписаться от уведомлений
search_house - Поиск дома по UNOM
search_coords - Поиск дома по координатам
stats - Статистика потребления дома
status - Статус системы
```
